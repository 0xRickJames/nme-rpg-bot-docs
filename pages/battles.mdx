# Battles

Initiate a battle between you and a rival or include several fighters for a free for all. Use your Armor, Weapons, Items, Spell and Skill to defeat your opponents.

![battle](/images/battle.png)

## 1v1

Use the command `/battle` and select one other Discord user in the OctoMob server to start a duel to the death.

## Group Free For All

Use the command `/battle` and select every other fighter you want involved in the free for all. Remeber, only one will survive.
